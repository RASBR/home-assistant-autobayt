# v0.1.2 Release - Bug Fix: Reload Integration/Device

## Summary
Version 0.1.2 fixes a critical bug related to reloading the integration or individual devices through Home Assistant's UI.

---

## Bug Fix

### Integration/Device Reload Issue
- **Fixed**: Resolved issue where reloading the integration or individual devices through Home Assistant's configuration UI could cause errors or unexpected behavior
- **Impact**: Users can now safely reload the integration or individual devices without encountering errors
- **Scope**: Applies to both integration-wide reload and per-device reload operations

---

## Compatibility
- **Home Assistant**: 2024.1.0 or later
- **Python**: 3.11 or later
